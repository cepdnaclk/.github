## ðŸ“ˆ Contribution Guidelines

Administration strongly advise to use this organization only for course projects and department-related repositories. Any other repositories should be created under personal accounts.

Organization members and contributors should follow these guidelines when adding new projects or contributing to existing repositories:

1. **Fork the relevant repository** or use a provided template.
2. Follow **naming conventions** for any course-specific project repositories:

```bash
eYY-CATEGORY-Project-Name
```

where `YY = batch`, and `CATEGORY` is the project tag. [Read More](https://projects.ce.pdn.ac.lk/docs/how-to-add-a-project/)

3. Write clear descriptions and documentation within your project repository.
4. Use issue threads for any questions or discussions related to specific repositories.
5. For any organizational or administrative questions, contact the GitHub Organization Administrator.
